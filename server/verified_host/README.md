# AGRI2RIST HUB - Verified Host Certification System (Dashboard module)

## Quickstart

1. Create virtualenv: python -m venv venv
2. Activate: .\venv\Scripts\Activate
3. Install: pip install -r requirements.txt
4. Run: python app.py
5. Open: http://localhost:5001/dashboard

This module uses SQLite for local testing. schema.sql contains MySQL-compatible schema.
